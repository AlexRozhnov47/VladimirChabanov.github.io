9. Простое наследование
Задача без теста

Создайте класс Food. Добавьте к класс:

  • закрытое поле calorie типа int;
  • Конструктор c параметром. Конструктор принимает значение которое должно быть установлено в поле calorie. Конструктор так же отображает в консоль строку: Калорийность продукта: {значение поля calorie};
  • Деструктор. Деструктор отображает в консоль строку: Продукт уничтожен!

Создайте класс Meat. Класс Food базовый для классаMeat.Добавьте к класс:

  • Конструктор c параметром. Конструктор принимает значение которое должно быть установлено в поле calorie. Конструктор так же отображает в консоль строку: Тип продукта: Мясо;
  • Деструктор. Деструктор отображает в консоль строку: Мясо уничтожено!
  • Метод с именем giveCalorie. Метод возвращает значение поля calorie и устанавливает его в нуль. Метод отображает на экран строку: Отдано {значение поля calorie} калорий.

Создайте класс Turtle. Класс Food базовый для классаTurtle.Добавьте к класс:

  • закрытое поле catchProbability типа int;
  • Конструктор c параметром. Конструктор принимает значение которое должно быть установлено в поле calorie. Поле catchProbability инициализируется случайный числом в диапазоне [0...100]. Конструктор так же отображает в консоль строку: Тип продукта: Черепаха;
  • Деструктор. Деструктор отображает в консоль строку: Черепаха уничтожена!
  • Метод с именем giveCalorie. Генерируется случайное число от 0 до 100. Если это число меньше catchProbability, то возвращает значение поля calorie, устанавливает его в нуль. Метод отображает на экран строку: Черепаха поймана. Отдано {значение поля calorie} калорий. Иначе возвращается нуль и на экран выводится строка: Черепаха убежала.
