Подготовка рабочего места

  1. Скачать и установить Java Development Kit;
  2. Прописать в системную переменную PATH путь к папке bin с установленным JDK. Например: C:\Program Files\Java\jdk1.8.0_181\bin

Правильность установки можно проверить следующим образом. Откройте окно команд и введите: java -version или javac -version.

Как написать Hello World на JAVA

Программа на JAVA представляет собой набор взаимодействующих классов. То есть нет процедур/функций как например в С++.

Исходные файлы программы на JAVA должны соответствовать следующим требованиям:

  1. Файл с исходным кодом - это обычный текстовый файл с расширением .java;
  2. Название файла с кодом должно совпадать с названием главного класса описанного в файле;
  3. Если в файле объявленно несколько классов, то только один (главный) может быть объявлен как public, видимость остальных классов будет ограниченна пакетом в котором расположен файл;

Структура простейшей программы

1
public class HelloWorld {
2
    public static void main(String[] args) {
3
        System.out.print("Hello, world!");
4
    }
5
}

В первой строке объявлен класс с именем HelloWorld для этого используется ключевое слово class. Экземпляры данного класса можно создавать в любом другом классе, т.к. он объявлен с использованием модификатора public.

Данный класс содержит метод main тип возвращаемого значени void, а в качестве параметров принимается массив строк String[] args. Модификатор public говорит, что данный метод можно вызывать в любом другом коде, в котормом доступен этот класс, а благодаря модификатору static для вызова метода не нужно создавать экземпляр класса. В фигурных скобках расположено тело метода.

Согласно правилам языка метод main явлется точкой входа в программу, он должен быть объявлен точно как в строке 2.

В третьей строке вызывается метод print объекта out. Данный метод позволяет выводить на консоль текстовые сообщения.

Компиляция из командной строки

JAVA является компилируемым языком, поэтому перед запуском программы её нужно скомпилировать. Для этих целей в JDK входит компилятор javac. Он преобразует файл с исходным кодом в файл с байт-кодом, при этом имя файл будет совпадать с именем исходного файла, а рисширение измениться на .class.

В случае если программа состоит из одного файла, то чтобы скомпилировать программу выполните следующие действия:

  1. Откройте окно команд;
  2. Перейдите в папку с исходным кодом;
  3. Наберите команду: javac HelloWorld.java. Это приведёт к помпиляции программы, в результате чего появится новый файл HelloWorld.class;

Запуск из командной строки

Введите команду: java HelloWorld. Несмотря на то, что файл с байт-кодом имеет расширение .class, для запуска расширение указывать не нужно.

Т.к. программа на JAVA это набор классов, то в каждом из них можно определить свой метод main со своей реализацией.

В отличае от других языков есть некоторые особенности при указании пути к запускаемому файлу. Например, если попытаться запустить программу таким образом:

xxxxxxxxxx
1
1
java bin\f1\Hello

то bin\f1 будет воспринято java машиной как указание на то, что класс Hello размещён в пакете bin\f1, но не как путь к файлу.

Чтобы запустить на выполнение сласс Hello находящийся в пути bin\f1 относительно текущего каталога, нужно воспользоваться флагом -classpath или сокращённой версией -cp. Например:

xxxxxxxxxx
1
1
java -classpath bin\f1\ Hello

Соглашение об именовании JAVA

JAVA как и С++ регистрозависимый язык. Ниже приведена таблица с описанием правил именования принятых в JAVA. Не соблюдение нижеперечисленных правил - не является ошибкой с точки зрения языка, но всё же настоятельно рекомендуется соблюдать данные праила, т.к. они де-факто являются стандартом.

Базовый синтаксис

Базовым элементом любой программы являются переменные. В JAVA переменные нужно объявлять явно с указанием типа. Все типы в JAVA делятя на два типа: примитивные и ссылочные.

Примитивные типы

Примитивных типов всего 8 штук:

  • boolean;
  • char;
  • byte, short, int, long;
  • float, double.

Переменная примитивного типа объявляется следующим образом:

boolean

В JAVA существует шесть операторов сравнения: ==, !=, >, >=, <, <=.

Доступные операции

К переменным целочисленного типа можно применять все стандартные арифметические операции: +, -, *, / (целочисленное), %. Доступны так же операторы инкремента и декремента в обоих формах: a++, --a.

Будьте внимательны, деление на нуль вызовет падение программы с ошибкой. Переполнение типа ошибкой не является.

Над целыми цислами можно выполнять не только арифметические операции, но и побитовые логические тоже:

char

С одной стороны со значениями типа char допускается выполнение всех вышеперечисленных операций, как и с любым целым числом.

С другой стороны, в отличае от целых чисел, char беззнаковый и хранит не просто число, а код символа в кодировке unicode, а точнее utf-16.

Символьные литералы хранятся в коде в виде символа заключённого в одинарные кавычки. Символ может быт записан либо явно, либо в виде эскейп последовательности:

Доступные операции

К вещественным числам применимы все теже арифметические операции, что и к целым : +, -, *, /, % (и эта тоже). Доступны так же операторы инкремента и декремента в обоих формах: a++, --a. Бинарные операции не поддерживаются.

При работе с вещественными числами есть некоторые особенности, о которых нужно знать:

Бесконечности и NaN являются полноправными значениями вещественного типа и ведут себя в соответствии с математическими аналогами.

При сравнении NaN есть одна особенность, приверка на равенство NaN с чем угодно, даже с самим собой, всегда даст false.

Ссылочные типы

Все остальные типы доступные в JAVA - ссылочные.

Переменная ссылочногоо типа представляет собой ячейку памяти содержащую ссылку на участок памяти в котором расположен объект.

xxxxxxxxxx
3
1

2

3

Из кода заметно, что в строках 1 и 2 создаётся сразу две сущьности - это переменная содержащая ссылку и сам объект. Можно создать переменную не ссылающуюся ни на какой объект, для этого используется ключевое слово null.

В отличае от примитивных типов код приведенный ниже вызовет не копирование объекта, а копирование ссылки на объект:

xxxxxxxxxx

1

2

А это значит, что и a и b дают доступ к одному и тому же объекту.
